# Юг-Стрим — лендинг

Одностраничный сайт оптового поставщика белорусских продуктов по Югу России.
React + Vite (без CDN/Babel в браузере — нормальная продакшен-сборка).

## Запуск

```bash
npm install        # один раз
npm run dev        # дев-сервер с горячей перезагрузкой → http://localhost:5173
npm run build      # продакшен-сборка в папку dist/
npm run preview    # локальный просмотр собранной версии
```

## Деплой

После `npm run build` папка `dist/` — это готовый статический сайт. Залейте её
содержимое на любой хостинг: Netlify, Vercel, GitHub Pages, Cloudflare Pages
или обычный nginx/Apache. Сборка использует относительные пути (`base: "./"`),
поэтому работает и из подпапки.

## Где что править

| Что | Файл |
|-----|------|
| Телефон, навигация, логотип | `src/sections/SectionsTop.jsx` |
| Каталог: категории и товары | `src/sections/SectionsMid.jsx` (`CATS`, `PRODUCTS`) |
| Цифры в блоке «Почему мы» | `src/sections/SectionsMid.jsx` (`STATS`) |
| Список заводов-партнёров | `src/sections/SectionsMid.jsx` (`BRANDS`) |
| Адреса магазинов, контакты, гео | `src/sections/SectionsBottom.jsx` |
| Логотип (картинка) | `src/assets/logo.png` |
| Цвета, шрифты, тени | `src/styles/base.css` (`:root`) |

### Число лет на рынке

В `src/sections/SectionsMid.jsx` есть константа `YEARS_ON_MARKET = null`.
Пока она `null`, вместо выдуманной цифры показывается достоверный факт про
автопарк. Поставьте реальное число (например `YEARS_ON_MARKET = 7`) — появится
блок «7+ лет на рынке».

## Подключение реальных заявок

Сейчас формы (звонок / прайс) работают как **заглушка**: показывают успех, но
никуда не отправляют. Вся логика отправки — в одном файле `src/lib/leads.js`.

Внутри функции `submitLead` уже заготовлены два варианта — **Telegram-бот** и
**POST на свой бэкенд/Formspree**. Чтобы включить нужный:

1. Откройте `src/lib/leads.js`, удалите блок-заглушку, раскомментируйте вариант.
2. Создайте файл `.env` в корне и пропишите ключи, например для Telegram:
   ```
   VITE_TG_TOKEN=123456:ABC...
   VITE_TG_CHAT_ID=123456789
   ```
3. `npm run build` — готово.

> Примечание: токены в `VITE_*` попадают в клиентский бандл и видны в браузере.
> Для боевого режима надёжнее принимать заявки своим бэкендом/прокси, а не слать
> запрос в Telegram напрямую из браузера.

## Структура

```
index.html            точка входа
src/
  main.jsx            рендер приложения, подключение стилей
  App.jsx             состояние, модалки, тосты
  icons.jsx           SVG-иконки
  components.jsx      переиспользуемые компоненты (форма, модалка, тост…)
  lib/leads.js        отправка заявок (сейчас заглушка)
  sections/           секции лендинга
  styles/             CSS (дизайн-система + секции)
  assets/             логотип
_legacy/              исходный прототип (CDN-версия), можно удалить
```
